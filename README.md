# FSD Lab – Single Page Applications (React)

This repository contains frontend lab experiments developed using React.

## Folder Structure

fsd-lab
├── experiment-1-counter
├── experiment-2-todo
├── experiment-3-theme-toggle
└── experiment-4-form

## Experiments

### Experiment 1: Counter SPA
- Increment and decrement counter using React state

### Experiment 2: To-Do List SPA
- Add and delete tasks dynamically

### Experiment 3: Theme Toggle SPA
- Switch between light and dark mode

### Experiment 4: Simple Form SPA
- Capture and display user input instantly

## How to Run Any Experiment

1. Go into the experiment folder  
2. Install dependencies  
3. Start the app  

```bash
npm install
npm start
